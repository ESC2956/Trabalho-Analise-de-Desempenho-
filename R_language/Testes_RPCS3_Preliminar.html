Testes_RPCS3_Preliminar

Descrição

O documento apresenta dados captados no emulador RPSC3, esses dados estão presentes em arquivos .csv; Para cada cenário de teste foram feitas X execuções essas que serão agrupadas por e serão calculadas média e desvio dos dados; A cena utilizada para captação dos dados foi uma que contém varios inimigos/entidades na tela. Jogo: The Last of Us Ambiente de teste: NixOS 25.11 CPU: Ryzen 9 9900X GPU: RX 7800 XT

Libraries usadas

library(readr)
library(ggplot2)
library(dplyr)

Leitura dos arquivos .csv e calcula da média e desvio

Lê todos os arquivos do mesmo cenario e experimento e faz a média dos valores utilizando “group_by(elapsed)”.

csv_avx_12_cores <- list.files(path = "./Experiments/12_cores", pattern = "*.csv", full.names = TRUE)
csv_avx_6_cores <- list.files(path = "./Experiments/6_cores", pattern = "*.csv", full.names = TRUE)
csv_avx_4_cores <- list.files(path = "./Experiments/4_cores", pattern = "*.csv", full.names = TRUE)

raw_data_12_cores <- do.call(rbind, lapply(csv_avx_12_cores, read.csv))
raw_data_6_cores <- do.call(rbind, lapply(csv_avx_6_cores, read.csv))
raw_data_4_cores <- do.call(rbind, lapply(csv_avx_4_cores, read.csv))

mean_data_12_cores <- raw_data_12_cores %>%
  group_by(elapsed_ms) %>%
  summarise(
    mean_frametime = mean(frametime_ms),
    deviation_frametime = sd(frametime_ms),
    mean_ppu = mean(ppu_usage),
    deviation_ppu = sd(ppu_usage),
    mean_spu = mean(spu_usage),
    deviation_spu = sd(spu_usage)
  )
mean_data_6_cores <- raw_data_6_cores %>%
  group_by(elapsed_ms) %>%
  summarise(
   mean_frametime = mean(frametime_ms),
    deviation_frametime = sd(frametime_ms),
    mean_ppu = mean(ppu_usage),
    deviation_ppu = sd(ppu_usage),
    mean_spu = mean(spu_usage),
    deviation_spu = sd(spu_usage)
  )
mean_data_4_cores <- raw_data_4_cores %>%
  group_by(elapsed_ms) %>%
  summarise(
   mean_frametime = mean(frametime_ms),
    deviation_frametime = sd(frametime_ms),
    mean_ppu = mean(ppu_usage),
    deviation_ppu = sd(ppu_usage),
    mean_spu = mean(spu_usage),
    deviation_spu = sd(spu_usage)
  )

Graficos

Comparação ppu_usage e spu_usage

# 12 cores
ggplot(mean_data_12_cores, aes(x = elapsed_ms)) +
  geom_line(aes(y = mean_ppu, color = "PPU"), linewidth = 1) +
  geom_point(aes(y = mean_ppu, color = "PPU"), size = 1.5) +
  geom_errorbar(aes(ymin = mean_ppu - deviation_ppu,
                    ymax = mean_ppu + deviation_ppu),
                color = "black", width = 0.2, alpha = 0.7) +
  geom_line(aes(y = mean_spu, color = "SPU"), linewidth = 1) +
  geom_point(aes(y = mean_spu, color = "SPU"), size = 1.5) +
  geom_errorbar(aes(ymin = mean_spu - deviation_spu,
                    ymax = mean_spu + deviation_spu),
                color = "black", width = 0.2, alpha = 0.7) +
  scale_color_manual(values = c("PPU" = "blue", "SPU" = "red")) +
  labs(title = "Uso de PPU e SPU ao longo do tempo",
       subtitle = "12 cores",
       x = "Tempo (ms)", y = "Uso (%)",
       color = "Componente") +
  scale_y_continuous(breaks = seq(0, 100, by = 5), expand = c(0, 0),limits = c(0,100)) +
  theme_minimal()

Graficos com média e desvio do frametime ao longo do tempo para diferentes números de cores e um graficos juntando dados para melhor visualização. Através disso fica claro que aumentar os cores impacta no frametime durante a execução de um jogo, também vemos que no decorrer do tempo por conta de o emulador se estabilizar vemos um pequeno declinio no frametime para todos os números de cores testados.
